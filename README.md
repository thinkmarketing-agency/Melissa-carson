# Melissa Carson – Realtor Website

## Asset Folder Structure

Drop your files into the `images/` folder with these exact filenames:

| File | Description |
|---|---|
| `images/headshot.jpg` | Melissa's headshot (square or portrait, min 300×300px) |
| `images/logo-stellar.png` | Stellar Realty NW logo (transparent PNG preferred) |
| `images/logo-team.png` | Mountain Sky Team logo (transparent PNG preferred) |

## Brand Colors (current)
- Navy: `#1a2b45`
- Gold: `#b8922a`
- Cream background: `#f7f4ef`

To change colors, edit the `:root` variables at the top of `css/style.css`.
